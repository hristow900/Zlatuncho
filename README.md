# Daily Reward Collector (Sellable Automation Tool)

Automation tool that logs into a website and collects daily rewards.
Includes:

- Flask dashboard
- Selenium automation
- SQLite logging
- Scheduler mode
- .env credentials protection
- Headless browser support

## Run

pip install -r requirements.txt
python app.py

## Features

- Manual trigger
- Auto daily schedule
- Logs dashboard
- Error screenshots ready integration

## Sellable Use Cases

- game reward automation
- website check-in automation
- daily login farming helper
- automation learning template
